# rust_js

[![Rust CI](https://github.com/isinghabhishek/rust_js/actions/workflows/quality-check.yml/badge.svg?branch=master)](https://github.com/isinghabhishek/rust_js/actions/workflows/quality-check.yml)
[![Release](https://github.com/isinghabhishek/rust_js/actions/workflows/release.yml/badge.svg)](https://github.com/isinghabhishek/rust_js/actions/workflows/release.yml)

A small Rust learning project comparing Rust concepts with JavaScript.

## CI

The GitHub Actions workflow checks formatting, lints with Clippy, and runs tests on pushes and pull requests.

## Releases

Push a semantic version tag such as `v0.1.0` to trigger the release workflow and publish a GitHub Release with the compiled Linux binary.
